
This piece of code was forked from:  https://github.com/andreiva/raspberry-pi-bme280

## Download

```
git clone https://github.com/wryan67/bme280_rpi_lib.git

```

## Dependencies
```
sudo apt-get install libi2c-dev i2c-tools wiringpi
```

## Compiling
```
make
```

## Install
```
sudo make install
```

## Demo
